Modificar y ajustar conjunto de datos

Antes de comenzar el anlisis de bivariante para el conjunto de datos que tenemos, es necesario modificar los datos para poder proceder a nuestro anlisis. Hemos dividido nuestro conjunto de datos en variables categricas y variables numricas. Para poder llevar a cabo el modelado estadstico, hemos convertido las variables categricas en factores y ahora podemos representar los datos categricos. Al convertir las variables categricas en factores, hemos predeterminado los valores que puede coger la variable y a estos valores se le refiere como niveles. En adicin, hemos corregido ciertos fallos en el conjunto de datos as como valores inadecuados para algunas variables. Para hacer estas correcciones, hemos eliminado algunos dgitos de los valores para las variables numricas en donde los errores eran muy claros. Para aquellos errores que son menos obvio, le hemos asignado NA. Por ltimo, para modelar todos los datos en la misma escala, hemos centralizado los datos numricos.

centralizados <- function(var_num){
  mean_var <-  mean(var_num, na.rm = T)
  var_cen <- var_num - mean_var
  var_sd <- sd(var_num, na.rm = T)
  var_cen_sd <- var_cen/var_sd
  return(var_cen_sd)
}   

for (i in names(data_num)){
  data_num[paste0(i,"_centralizados")] <- centralizados(data_num[,i])
}

Modelos lineales generalizados para variables numricas

Los modelos lineales son utilizados para observar el efecto que tiene la variable independiente en la variable dependiente e identificar si influye de manera estadsticamente significativa. Es por eso que hemos realizado modelos lineales generalizados para la variable dependiente (que en este caso es si el paciente est enfermo del corazn o no) en funcin de las variables numricas que son las variables independientes. Luego de adquirir los valores p, los hemos representado en un barplot para poder visualizarlos. Podemos determinar que las variables correpondiendo a ca, oldpeak, thalach y age influyen de manera estadsticamente significativa en la enfermedad del corazn de los pacientes ya que tienen un valor p menor de 0,05 mientras las variables chol y trestbps no influyen de manera estadsticamente significativa.

mis_pvalores_var_num <- NULL
for (i in 7:12){
  modelos_var_num <- glm(data_num[,13] ~ data_num[,i], family = binomial(link = "logit"))
  mis_pvalores_var_num <- c(mis_pvalores_var_num,(summary(modelos_var_num)$coefficients[2,4]))
  
}

nombres_pvalores <- colnames(data_num[1:6])

barplot(mis_pvalores_var_num, horiz = TRUE, names = nombres_pvalores,cex.names = 0.6, las = 1, col = 'blue', xlab = "Valor p")

Modelos lineales generalizados para variables categricas

Cuando hacemos un modelo lineal generalizado y consideramos la interaccin de dos variables categricas, la visualizacin es un poco ms compleja. Como estamos analizando variables categricas, hay que considerar la manera en que influye cada nivel en cada variable. Se considera estadsticamente significativo si el valor p es menor de 0,05 y podemos observar que de todas los valores p adquiridos de los modelos lineales generalizados, solo los niveles de reversable defect(thal), flat(slope), abnormalities(ecg) y yes(fbs) no tienen un valor p menor de 0,05. Esto significa que estos niveles de sus variables correspondientes no influyen de manera estadsticamente signficativa en la enfermedad del corazn del paciente y que los otros niveles s influyen de manera estadsticamente significativa.
