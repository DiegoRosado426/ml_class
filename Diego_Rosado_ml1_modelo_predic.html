Introduccin al dataset

El dataset utilizando para crear este modelo predictivo proviene de un conjunto de datos creado por Dr.William H. Wolberg para el cancer de mama en University of Wisconsin Hospitals, Madison. Las muestras llegaban de manera peridica mientras Dr.Wolberg haca reportajes sobre sus casos clnicos. Este conjunto de datos tiene un total de 699 instancias donde cada una contiene once atributos. Por ltimo, nuestra variable target para este conjunto de datos es Class que es una variable categrica compuesta por dos niveles, Benign y Malignant. A travs de los valores que le corresponden a las variables predictoras, podremos determinar el valor de la variable target que nos dice si el tumor es benigno o maligno.

Antes de poder crear nuestros modelos para el conjunto de datos, es necesario limpiar y modificar los datos para que se puedan utilizar de una manera adecuada. Para este archivo que contiene los datos, hemos eliminado los separadores que dividen cada columna que en el caso de los datos de training eran h_. Luego hemos creado un data frame con los datos y le hemos asignado el nombre correspondiente a cada columna. En la base de datos, se encontraban valores con signos interrogativos que identificaban los valores NA y los hemos identificado de esa manera en el data frame. Adicionalmente, haba una cantidad de valores que no caan dentro del rango de valores posibles para cada variable y se tomaron las decisiones adecuadas para poder incluir estos valores. Por ejemplo, como los valores posibles para cada variable van del 1-10, si apareca un valor igual a 20, se le asignaba el valor de 2 y as hicimos con los otros valores inapropiados en el dataset. Por ltimo, hemos clasificado aquellas variables que van desde la columna 1 hasta la 10 como variables numricas y aquellas en las columnas 11 y 12 como categricas. Las columnas 11 y 12 le corresponden a Group y Class, respectivamente.

Modelos lineales generalizados para variables numricas

Los modelos lineales son utilizados para observar el efecto que tiene la variable independiente en la variable dependiente e identificar si influye de manera estadsticamente significativa. Es por eso que hemos realizado modelos lineales generalizados para la variable dependiente (que en este caso es si el paciente tiene un tumor maligno o benigno) en funcin de las variables numricas que son las variables independientes.Para estos modelos, hemos considerado un valor p significativo como uno menor de 0,05. Luego de adquirir los valores p de los modelos, cada uno sali estadsticamente significativo.

mis_pvalores_var_num <- NULL
for (i in 2:10){
  modelos_num <- glm(mis_datos[,12] ~ mis_datos[,i], family = binomial(link = "logit"))
  mis_pvalores_var_num <- c(mis_pvalores_var_num,(summary(modelos_num)$coefficients[2,4]))
  
}
nombres_variables <- names(mis_datos[2:10])

Modelo predictivo

El ltimo paso de nuestro modelo predictivo es hacer nuestras predicciones. Utilizando el model multivariante ptimo con las variables predictoras adecuadas. Hemos usado la tcnica de cross-validation en donde evaluamos la eficencia de nuestro modelo ya que es entrenado por un subconjunto de nuestra base de datos y luego hacemos el testing con datos que el modelo an no ha visto. Luego de correr el modelo varias veces, he determinado que el accuracy de nuestro modelo tiene una media de 0.96.
